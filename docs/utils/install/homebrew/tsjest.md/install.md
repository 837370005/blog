# 基于TypeScript, Jest
### 创建初始化项目
```js
mkdir coding-ts && cd coding-ts
npm init -y
```
### 安装ts的开发环境依赖
```js
npm install typescript ts-node @types/node -D
```

### 安装babel的开发环境依赖
babel 是一种源码到源码的转换编译器，是一个node命令行工具，它的作用是对我们源代码进行转码（把es6=>es5）
```js
npm install babel-jest @babel/core @babel/preset-env @babel/preset-typescript -D
```
### 配置babel.config.js
```js
module.exports = {
  presets: [
    ['@babel/preset-env', {targets: {node: 'current'}}],
    '@babel/preset-typescript',
  ],
};
// 具体的参考这里，https://jestjs.io/docs/getting-started
```
### 安装jest的开发环境依赖
```js
npm install jest ts-jest @types/jest -D

```

jest会识别包含.test, spec等文件，或文件夹下的文件
新建sum.ts
```js
function sum(a: number, b: number) {
    return a + b
}
export default sum
```
新建index.test.ts
```js
import sum from './sum'
test('adds 1 + 2 to equal 3', () => {
    expect(sum(1, 2)).toBe(3);
});

```
### 配置package.json的相关script
```js
"scripts": {
    "build": "tsc",
    "test": "jest",
    "test:help": "jest --help",
    "test:debug": "jest --debug",
    "test:verbose": "jest --verbose",
    "test:init": "jest --init",
    "test:json": "jest --json",
    "test:exportJson": "jest --json --outputFile=./export/reporter.json",
    "test:coverage": "jest --coverage",
    "test:watchAll": "jest --watchAll",
    "pretty": "prettier --config .prettierrc \"code/**/*.ts\" --write",
    "pretty-watch": "onchange \"code/**/*.ts\" -- prettier --write {{changed}}",
    "git:pull": "pull",
    "git:push": "push"
  }

```
